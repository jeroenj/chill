Chill by redstorm

Test RESTful services the easy way

Install

  • Download or build a binary
  • Drag the application to any location you want
  • Run the application

Dependencies

  • MacRuby 0.5
  • Xcode 3.2 (in order to build it)
  • Mac OSX 10.6 Snow Leopard (64 bit enabled)

Download

  • chill-0.1.0.zip (MacRuby not embedded)
  • chill-0.1.0-macruby-embedded.zip (MacRuby embedded)
  • chill.zip (sources)
  • chill.tar.gz (sources)

Build it

  • You can also build the project by cloning it with Git: 
    $ git clone git://github.com/jeroenj/chill
  • Then open the Project with Xcode
  • Build it as release
  • Build the Compile & Embed target to embed MacRuby in the app. (This step is not required if you have MacRuby installed on the machine where you are going to run the applicatin)
get the source code on GitHub : jeroenj/chill
